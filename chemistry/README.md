# Chemistry - Learning from Scratch

## Curriculum Overview

This is a ground-up chemistry course structured as modular, incremental lessons.
Each module builds directly on the previous ones.

| Module | Topic | Status |
|--------|-------|--------|
| 1 | What is Matter? | Not started |
| 2 | The Atom | - |
| 3 | The Periodic Table | - |
| 4 | Chemical Bonding | - |
| 5 | Chemical Reactions | - |
| 6 | The Mole & Stoichiometry | - |
| 7 | States of Matter & Intermolecular Forces | - |
| 8 | Solutions & Acids/Bases | - |
| 9 | Thermochemistry | - |
| 10 | Intro to Organic Chemistry | - |

## How to Use
- Work through modules in order
- Each module folder contains lessons, simulations, and exercises
- Run Python scripts for interactive simulations
